## Build

1. `git clone`这个仓库
2. 安装nodejs hexo等环境
   - node 18.13.0
   - `npm install -g hexo-cli`
   - `npm install`  安装依赖

## 使用

- hexo g 生成静态文件
- hexo g -d 生成并部署
- hexo s -g 本地预览
- `hexo douban -bgm` 豆瓣
