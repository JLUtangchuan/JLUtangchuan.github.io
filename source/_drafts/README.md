#### 为什么要读书？

> 在阅读中进行反思；在阅读中分析

#### 小说和影视作品的欣赏

> 优秀的作品吸引人阅读和思考，每个人都可以从其中获得不同的感悟；我们有时会与其中人物的性格产生共鸣，引起深入的思考。此外其中句子的积累也很有必要，有些句子往往在我们读完整篇故事才能理解，这些句子也最能表达作者想传递的感情。
